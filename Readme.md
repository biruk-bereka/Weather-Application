# 📖 [Weather Application] <a name="about-project"></a>

**Weather Application** is is a website in which users can view weather condition of different cities and weather details for each cities. Users can search by city name. 

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

<details>
  <summary>Client</summary>
  <ul>
    <li><a href="https://reactjs.org/">React.js</a></li>
    <li><a href="https://redux-toolkit.js.org/">Redux.js</a></li>
  </ul>
</details>

<!-- Features -->

### Key Features <a name="key-features"></a>

  - Set  up project with create-react-app
  - Create home and details routes 
  - Fetch weather data from openWeatherMap end point
  - Display weather data in home and weather details in details component 
  - Implement search functionality

### Prerequisites

In order to run this project you need:

```sh

```
### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/biruk-bereka/weather-application.git
```
-

### Install

Install dependencies: 

```sh
npm i
```

### Usage

To run the project, execute the following command:

```sh
npm start
```
### Run tests

To run tests, run the following command:

```sh
npm test
```

To run the stylehint linter
